
## What Is "From Strehl to Kullback-Leibler"
A Unified Fidelity Framework for Stationary, Event-Driven, and Quantum Imaging Optics:
Detector-protocol-aware lens design on the eikonal bridge between classical and quantum imaging.

This repository hosts companion materials for the perspective paper From Strehl to Kullback-Leibler. 
The work proposes a 2x2 typology of fidelity measures for optical lens design — Strehl ratio (classical / stationary), 
normalized Event-MTF (classical / event-driven), quantum state fidelity (quantum / stationary), and 
a Kullback-Leibler-based fidelity on photon-arrival distributions (quantum / event-driven) — and 
shows that existing tools (CODE V, Zemax, LightTools, RSoft) need new merit-function operands, not new physics.

Core theme: when the detector changes, the lens merit function must change with it.
